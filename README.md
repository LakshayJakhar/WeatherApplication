## Project Overview
WeatherApp is a **C# Windows Forms application** that allows users to enter a city name and retrieve real-time weather data using the OpenWeatherMap API. The app shows:

- Current temperature (in Celsius)  
- Weather condition description  

This project demonstrates **API integration, asynchronous programming, and GUI design** in a desktop environment — making it a strong portfolio project for beginners to intermediate developers.

---

## Features
- Input city name to get weather data  
- Displays **temperature in Celsius**  
- Shows **weather condition description**  
- **Async HTTP requests** for smooth performance  
- Handles invalid input and errors gracefully

## Technologies Used
- **C#**  
- **.NET Windows Forms**  
- **Newtonsoft.Json** (for JSON parsing)  
- **OpenWeatherMap API**
